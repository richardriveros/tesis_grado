# Tesis de Grado - Repositorio de Arqueo de Fuentes

Colección de Tesis de Grado de Doctorados con un sistema completo de arqueo (documentación) de fuentes bibliográficas.

## Propósito

Este repositorio proporciona:
1. Una colección organizada de tesis doctorales
2. Un sistema de arqueo (registro) de fuentes bibliográficas
3. Trazabilidad completa entre tesis y sus fuentes
4. Índices y búsqueda de fuentes por autor y tema

## Estructura del Repositorio

```
tesis_grado/
├── README.md                 # Este archivo
├── tesis/                    # Directorio de tesis doctorales
│   ├── README.md            # Guía de tesis
│   └── T001_Ejemplo_2026/   # Ejemplo de tesis
├── fuentes/                  # Materiales de fuentes
│   ├── README.md            # Guía de fuentes
│   └── libros/              # Organizados por tipo y año
└── arqueo/                   # Sistema de arqueo de fuentes
    ├── README.md            # Guía del arqueo
    ├── arqueo_fuentes.md    # Registro principal
    ├── plantilla_fuente.md  # Plantilla para nuevas fuentes
    ├── indice_por_autor.md  # Índice por autor
    └── indice_por_tema.md   # Índice por tema
```

## Guía Rápida

### Para Consultar Fuentes

1. Vea el **[Registro Principal de Fuentes](./arqueo/arqueo_fuentes.md)** para todas las fuentes documentadas
2. Use el **[Índice por Autor](./arqueo/indice_por_autor.md)** para buscar por autor
3. Use el **[Índice por Tema](./arqueo/indice_por_tema.md)** para buscar por área temática

### Para Agregar una Nueva Tesis

1. Lea la **[Guía de Tesis](./tesis/README.md)**
2. Cree un directorio con formato `T###_Apellido_Año`
3. Use el ejemplo en `tesis/T001_Ejemplo_2026/` como plantilla
4. Complete toda la información requerida
5. Registre las fuentes utilizadas en el arqueo

### Para Agregar una Nueva Fuente

1. Lea la **[Guía del Arqueo](./arqueo/README.md)**
2. Consulte la **[Plantilla de Fuente](./arqueo/plantilla_fuente.md)**
3. Agregue el registro al archivo `arqueo/arqueo_fuentes.md`
4. Actualice los índices por autor y tema
5. Opcionalmente, agregue materiales en el directorio `fuentes/`

## Sistema de Identificación

### Tesis
- **Formato**: T### (T001, T002, T003, etc.)
- **Directorio**: `T###_ApellidoAutor_Año`
- **Ejemplo**: T001_Garcia_2024

### Fuentes
- **Formato**: F### (F001, F002, F003, etc.)
- **Tipos**: Libros (L), Artículos (A), Tesis (T), Conferencias (C), Web (W), Técnicos (D)
- **Directorio**: `tipo/año/F###_Apellido_PalabraClave`
- **Ejemplo**: libros/2014/F001_Hernandez_Metodologia
