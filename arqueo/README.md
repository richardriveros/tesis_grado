# Directorio de Arqueo de Fuentes

Este directorio contiene la documentación del arqueo de fuentes bibliográficas utilizadas en las tesis de grado de doctorado.

## Contenido

- **arqueo_fuentes.md**: Registro principal de todas las fuentes documentadas
- **plantilla_fuente.md**: Plantilla para registrar nuevas fuentes
- **índice_por_autor.md**: Índice organizado por autor
- **índice_por_tema.md**: Índice organizado por área temática

## Cómo Usar

1. Consulte `arqueo_fuentes.md` para ver todas las fuentes registradas
2. Use `plantilla_fuente.md` como guía para agregar nuevas fuentes
3. Mantenga los índices actualizados cuando agregue nuevas fuentes

## Estructura de Archivos

```
arqueo/
├── README.md                 # Este archivo
├── arqueo_fuentes.md        # Registro principal
├── plantilla_fuente.md      # Plantilla para nuevas entradas
├── índice_por_autor.md      # Índice alfabético por autor
└── índice_por_tema.md       # Índice por área temática
```
