# Directorio de Tesis

Este directorio contiene las tesis de grado de doctorado de la colección.

## Estructura de Archivos

Cada tesis debe organizarse en su propio subdirectorio con el siguiente formato:

```
tesis/
├── T001_Apellido_Año/
│   ├── README.md              # Información de la tesis
│   ├── tesis.pdf             # Documento de la tesis
│   ├── resumen.md            # Resumen ejecutivo
│   ├── fuentes_citadas.md    # Lista de fuentes citadas
│   └── anexos/               # Materiales adicionales
└── T002_Apellido_Año/
    └── ...
```

## Nomenclatura

- **ID de Tesis**: T### (T001, T002, T003, etc.)
- **Nombre de Directorio**: T###_ApellidoAutor_Año
  - Ejemplo: `T001_Garcia_2024`

## Información Requerida por Tesis

Cada tesis debe incluir un README.md con:

1. **Información Básica**
   - Título completo
   - Autor(a)
   - Director(a) de tesis
   - Año de presentación
   - Universidad/Institución

2. **Resumen**
   - Resumen de la investigación
   - Palabras clave
   - Área temática

3. **Fuentes Utilizadas**
   - Lista de IDs de fuentes del arqueo
   - Referencias bibliográficas completas

4. **Estado**
   - En progreso / Completada / Publicada

## Ejemplo

Ver el directorio `T001_Ejemplo_2026/` para una estructura de referencia.

## Cómo Agregar una Nueva Tesis

1. Cree un nuevo directorio siguiendo la nomenclatura
2. Copie la plantilla de README.md
3. Complete toda la información requerida
4. Agregue el documento de la tesis (PDF)
5. Registre las fuentes utilizadas en el arqueo
6. Actualice este índice
