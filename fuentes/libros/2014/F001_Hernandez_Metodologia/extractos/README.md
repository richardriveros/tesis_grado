# Extractos y Citas - F001

## Instrucciones

Este directorio contiene extractos breves y citas textuales de la fuente para propósitos de referencia académica. Solo se incluyen fragmentos relevantes bajo uso justo para citación académica.

## Formato de Extractos

Cada extracto debe incluir:
- Página(s) de origen
- Contexto del extracto
- Cita textual (entre comillas)
- Uso previsto en tesis
